# Bilibili-Infochcker
核心功能目前包括：输入网页视频或音频链接后，自动提取音频；用本地 whisper.cpp 转成文字；再用 AI 做断句、标点和保守纠错；最后结合联网搜索结果，对视频里的关键主张做结构化核查，输出真实性、时效性、专业性、娱乐性和情绪引导嫌疑等评分，并生成 Markdown 和 JSON 报告。 技术路径上，主程序用 C# + WPF 做成轻量 Windows 桌面工具；音视频提取走 yt-dlp + ffmpeg；语音识别走本地 whisper.cpp；AI 评价部分拆成独立模块，负责搜索规划、联网取证和结构化评分。这样既能保持桌面端足够轻，也方便后续替换模型或扩展新的核查逻辑。
